    Usage

    In the sentence he\'s behaving like he owns the place, like is a conjunction meaning \xe2\x80\x98as if\xe2\x80\x99, a usage regarded as incorrect in standard English. Although like has been used as a conjunction in this way since the 15th century by many respected writers, it is still frowned upon and considered unacceptable in formal English, where as if should be used instead
