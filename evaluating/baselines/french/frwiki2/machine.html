Machine (homonymie)

Cette page d’homonymie répertorie les différents sujets et articles partageant un même nom.

Sur les autres projets Wikimedia :

  • machine, sur le Wiktionnaire

Selon le contexte machine peut désigner :

  • une machine, un dispositif capable d'effectuer des travaux pour lequel il a été pensé et manufacturé ;
  • machine abstraite.
  • Machine, un roman de l'écrivain japonais Riichi Yokomitsu.
  • Machines, un jeu vidéo publié en 1999.
  • En sport collectif Machine peut désigner le joueur le plus complet, qui allie une grande puissance, une grande technique et une intelligence stratégique hors-normes, tout cela combiné à une humilité le rendant parfait. Ce sobriquet est apparu avec l'émergence de Piou, la machine de la section rugby auquel il appartenait.
