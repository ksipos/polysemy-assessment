Anatomie[modifier | modifier le code]

En anatomie, la main est la partie du membre supérieur humain située à l’extrémité de l’avant-bras. En référence à celle-ci :

  • En métrologie, la main est une unité de mesure du système impérial d'unités.
  • Dans un jeu de cartes, la main est l'ensemble des cartes tenues par un joueur (par exemple au poker)
  • La main est un hiéroglyphe égyptien

Toponyme[modifier | modifier le code]

  • Ma'in, parfois orthographié Main ou Maïn, était un petit royaume d'Arabie (territoire de l'actuel Yémen) dans les derniers siècles avant l'ère chrétienne.
  • Main est une station aérienne du métro de Chicago (États-Unis).
  • Main Street (rue principale ou grande rue) est un odonyme très courant dans le monde anglophone.

Hydronyme[modifier | modifier le code]

  • Le Main est une rivière d'Allemagne.
  • La Rivière Main est une rivière de Terre-Neuve-et-Labrador.

Patronyme[modifier | modifier le code]

  • Main, évêque de Rennes.
  • Albert Russell Main, zoologiste australien.
  • Grant Main, médaillé d'or olympique canadien en Aviron.
  • Kent Main, coureur cycliste sud-africain.
  • Lorne Main (1930-20191, joueur de tennis canadien.
  • Marjorie Main, actrice américaine.
  • Robert Main, astronome anglais.
  • Stephanie Main, patineuse artistique britannique
  • Terence Main, sculpteur américain.

Littérature - Fiction - Cinéma[modifier | modifier le code]

  • La Main est une nouvelle de Guy de Maupassant publiée en 1883.
  • La Main (The Hand), une secte ninja fictive dans les Marvel Comics.
  • La Main, film muet français réalisé par Henry Bérény (1909).
  • La Main, court métrage d'animation tchécoslovaque de Jiří Trnka (1965).
  • La Main, film français réalisé par Henri Glaeser (1969).

Autres[modifier | modifier le code]

  • En informatique, le mot anglais main (en français « principal ») est le nom couramment donné à la fonction principale d'un programme dans son code source.
  • En papeterie, la main (papeterie) a deux sens :
    • un groupe de 25 feuilles de papier ;
    • le type de retour tactile d'un papier, sa « raideur », mesurée par le rapport entre l’épaisseur de la feuille et son grammage
  • Les Mains d'Anvers, pâtisserie belge.
