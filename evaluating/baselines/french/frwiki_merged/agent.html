Noms communs[modifier | modifier le code]

Le terme agent désigne une entité (une personne ou un phénomène) qui a une action déterminante.

  • un agent, dans la fonction publique, est une personne chargée de gérer, d'administrer les affaires d'une administration publique, de l'État, d'une collectivité territoriale, ou d'un centre hospitalier. Dans ce sens, un agent peut :
    • disposer d'un statut de fonctionnaire, il est alors titulaire.
    • être embauché avec un contrat de travail, généralement à durée déterminée ; il est alors contractuel.
  • un agent commercial, dans le monde du commerce, est un intermédiaire agissant pour le compte d'un fournisseur auprès de clients;
  • un agent diplomatique, en diplomatie ;
  • un agent économique, en économie, est l'unité de base de l'analyse microéconomique ;
  • un agent, en linguistique, est un actant qui effectue l'action.
  • un agent cognitif, ou un système multi-agents, en sciences cognitives ;
  • un agent de communication, en communication, est une personne physique ou morale spécialisée dans la communication, ou bien un élément du modèle de communication de Jakobson.
  • un agent d'assurance, en assurance ;
  • un agent immobilier, dans l'immobilier ;
  • un agent secret[1], ou simplement « agent », dans le monde du renseignement, désigne selon la terminologie des services de renseignement, un individu fournissant des renseignements à la demande de son officier traitant.
  • un agent provocateur, en criminologie ;
  • un agent orange, en chimie, est à la fois un herbicide et une arme chimique ;
  • un agent de joueurs, en sport, est un intermédiaire entre un ou plusieurs joueurs et les clubs.
  • un agent littéraire est un agent qui représente les écrivains et gère les droits de leurs œuvres.
  • un agent social, en sociologie, est un acteur qui intervient dans un champ social.
  • un agent, en astrologie chinoise, est le nom donné aux éléments associés aux signes zodiacaux ("HSING" en chinois).
  • En médecine :
    • Agent alkylant
    • Agent chimiothérapeutique
  • En informatique, un agent est un processus :
    • Agent (Informatique)
    • Agent intelligent ou agent Virtuel
    • Agent mobile
    • User-Agent
    • Système multi-agents

Noms propres[modifier | modifier le code]

Saint chrétien[modifier | modifier le code]

  • Agent (VIe siècle), avec saint Pient et sainte Colombe, ermites à Moyenvic au diocèse de Nancy ; fêtés le 30 octobre[2].

Jeux vidéo[modifier | modifier le code]

  • Agent est un projet de jeu, annulé, de Rockstar North.

Bande dessinée[modifier | modifier le code]

  • Agent 212 est une série de bande dessinée humoristique belge.

Cinématographie[modifier | modifier le code]

  • Agent secret, (en anglais Sabotage) est un film américain de 1936, réalisé par Alfred Hitchcock, tiré du roman The Secret Agent.
  • Agents secrets est un film français réalisé par Frédéric Schoendoerffer et sorti le 31 mars 2004.
  • Agent trouble est un film de Jean-Pierre Mocky.
  • Triple agent est un film français d'Éric Rohmer sorti en 2004.

Entreprise[modifier | modifier le code]

  • Agent Provocateur est une entreprise britannique de lingerie.
