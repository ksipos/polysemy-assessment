Noms communs[modifier | modifier le code]

Sur les autres projets Wikimedia :

  • château, sur le Wiktionnaire
  • un château, construction médiévale ;
  • le château, superstructure d'un bateau ;
  • les châteaux, certains domaines viticoles, même si le domaine ne possède pas de construction de type « château » à proprement parler ;
  • le château de plaisance, résidence aristocratique qui a perdu ses caractéristiques défensives.

Patronymie[modifier | modifier le code]

Château est un nom de famille notamment porté par :

  • Agnès Château, actrice française ;
  • Dominique Chateau (1948-), philosophe français ;
  • Jean Château (1908-1990), pédagogue et professeur d'université français ;
  • Karl Chateau[1] (1992-), joueur français de rugby à XV ;
  • François Château (1886-1965), homme politique français ;
  • René Château (1906-1970), philosophe et homme politique français, connu également sous le nom de plume Jean-Pierre Abel ;
  • René Chateau[1], éditeur vidéo français indépendant.
