Sémantique et étymologie[modifier | modifier le code]

Dérivé du participe passé du verbe sourdre, le mot Source signifie l'origine, le principe, la cause. Actuellement, et depuis la fin du XVIIe siècle, (Dictionnaire français de Pierre Richelet, 1680), source désigne, conformément à l'une de ses origines latines scaturigo, de l'eau qui sort naturellement de terre, ou par métonymie le point où cette eau jaillit : fontaine, du latin fons.

Par métaphore, ce terme est aussi employé pour puits, trésor dans de nombreuses acceptions :

  • une source d'énergie ;
  • une source d'ions ;
  • une source de carbone ;
  • une source du droit ;
  • etc.

Néanmoins, de son sens premier : origine, principe, il a conservé toutes les significations :

  • Commencement, endroit (vieux), point (de départ), fondation (architecture), Abondance.
    • Rechercher l'origine d'un mot en remontant jusqu'à la source (Étymologie).
    • Art qui se rajeunit en se retrempant à ses sources.
    • Retour aux sources.
    • L'ambition, source de désordres.
    • sources des colonnes d'un temple.
  • Cause, ferment, générateur, germe, principe.
    • La révolte, source de vraie vie.
    • Mère : Prendre source, avoir sa source dans…
    • Sortir (de) : Gourmandise, immoralisme : Être une source d'ennuis, de plaisir.
  • Créer, produire : Le droit universel, source de toutes les lois positives.
  • Base, fondement : Aucune action humaine n'a de source unique.
  • Cause : Les Sources de la morale et de la religion.

Il existe aussi plusieurs sens spécialisés.

Information[modifier | modifier le code]

Une source désigne l'origine humaine d'une information ou l'origine technique d'une donnée. Elle peut être :

  • primaire si elle présente des faits ;
  • secondaire si elle analyse des sources primaires ;
  • tertiaire si elle compile des sources primaires et secondaires.

Sciences[modifier | modifier le code]

Hydrologie[modifier | modifier le code]

  • Une source est une eau qui sort naturellement de terre, ou par métonymie le point où cette eau jaillit.
  • Une source chaude est une source dont l'eau sort à une température supérieure à la température ambiante.

Physique[modifier | modifier le code]

  • En optique, une source lumineuse est l'origine d'où émanent les rayons d'un corps radiant ou réfléchissant ;
  • en thermodynamique, une source chaude et une source froide sont nécessaires au fonctionnement d'une machine thermique ;
  • en électromagnétisme et dans ses applications (télécommunications, astrophysique…), une source électromagnétique est un objet à partir duquel est émise une onde électromagnétique : émetteur radio, émetteur de télévision, satellite, corps céleste, etc. ;
  • en astrophysique, une source X ultralumineuse est un astre lumineux dans le domaine des rayons X.

Mathématiques[modifier | modifier le code]

  • En théorie des ensembles, la source d'une fonction est un ensemble auquel appartient la variable de la fonction ;
  • en théorie des graphes, une source est, dans la théorie des graphes, un sommet particulier d'un graphe orienté.

Informatique[modifier | modifier le code]

  • Le code source est un code lisible par un humain qui doit être traduit pour le rendre exécutable.
  • Les polices Source sont une famille de polices de caractères libres (Source Sans Pro, Source Code Pro et Source Serif Pro).

Technique[modifier | modifier le code]

Électronique[modifier | modifier le code]

  • La source est une des électrodes d'un transistor à effet de champ ;
  • un source commune est un type d'amplificateur électronique ;
  • une source d'alimentation électrique est un dispositif pouvant fournir ou évacuer de l'énergie électrique, sous la forme d'une source de tension ou d'une source de courant ;
  • une source d'une chaîne hi-fi.

Acoustique[modifier | modifier le code]

Une source linéaire acoustique est un réseau d'enceintes acoustiques.
